# How It Works 动画 — 完整提示词汇总

## 概览

- **静态帧**：20张，由 Nano Banana 生成
- **视频段**：10段，由即梦首尾帧插值生成

## 即梦运动提示词

> 每段上传对应首帧+尾帧，将以下文字填入即梦的运动描述框。
>
> **通用防失真原则（每段均适用）**：
> - 所有未被提及移动的物体，保持原位固定，不消失、不变形、不缩放
> - 角色仅做已描述的肢体动作，面部比例和轮廓不变形
> - 不生成粒子、火焰、烟雾、爆炸、强光扩散等特效
> - 所有运动速度缓慢匀速，无突然跳变

---

**段01｜01-A → 01-B**（建议时长：3–4秒）
```
角色双手缓慢向下落至键盘，手指逐一触碰按键，做打字动作。
显示器画面内，表单各字段从上到下依次填入文字，每行文字平稳出现，
屏幕整体比例不变，显示器不移动。
屏幕右下角铜色发送按钮从无到有以淡入方式出现，
颜色变为铜色，亮度仅轻微提升，无光晕向外扩散，无闪烁。
角色上身向前微倾约5度，椅子、桌面、键盘保持完全静止。
角色脸部比例和形状不变形。
```

**段02｜02-A → 02-B**（建议时长：3–4秒）
```
信封图标沿右上方45度轨迹匀速飞行，在2秒内飞出画面右上角边缘，消失。
【鼠标锁定】鼠标保持在桌面原位完全静止，不跟随手移动，不消失，不变形。
角色右手缓慢松开鼠标，手指伸直，手臂沿桌面水平向后收回至大腿上方停止。
角色上身缓慢向后靠入椅背，移动幅度约10度，放松姿态。
显示器屏幕变为接近空白状态，画面颜色均匀淡出，无闪烁。
画面右侧一个铜色圆形徽章（内含时钟图标）以淡入方式出现，
出现过程均匀，无缩放弹跳，无发光扩散。
显示器、键盘、桌面其余物品保持固定。
```

**段03｜03-A → 03-B**（建议时长：3–4秒）
```
角色右手缓慢将手中金属笔向下移至桌面，轻放在桌上，笔静止后不再移动。
纸箱保持在桌面原位，箱内剩余金属物件不移动、不消失。
桌面上各金属物件保持各自位置静止。
角色左手缓慢从桌边拾起一张小白卡，卡片移至胸前高度停止。
角色上身缓慢向后微靠，幅度约8度，眼神向下看向手中卡片。
```
