const PRODUCT_OPTIONS = [

  { value: 'Castro Catchall Tray', label: 'Castro Catchall Tray' },

**?product= pre-fill logic (per CONTEXT.md decisions):**
- Read `useSearch({ from: '/{-$locale}/contact' })` — extract `product` param
- If `product` matches a PRODUCT_OPTIONS value exactly → use that as defaultValues.productInterest
- If `product` is set but no match → defaultValues.productInterest = 'Other / Multiple Products'
- If no `product` param → defaultValues.productInterest = '' (Select shows placeholder)

**useForm setup:**
```typescript
const form = useForm({
  defaultValues: {
    contactName: '',
    companyName: '',
    role: '',
    email: '',
    phone: '',
    productInterest: initialProduct,  // derived from ?product= above
    quantity: '',
    timeline: '',
    message: '',
    website: '',  // honeypot
  },
  validators: { onSubmit: inquiryInsertSchema },
  onSubmit: async ({ value }) => {
    setSubmitError(null)
    try {
      const res = await fetch('/api/inquiry', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(value),
      })
      if (res.ok) {
        setIsSuccess(true)
      } else {
        setSubmitError('Something went wrong. Please try again or email us directly.')
      }
    } catch {
      setSubmitError('Something went wrong. Please try again or email us directly.')
    }
  },
})
```

- `const [isSuccess, setIsSuccess] = useState(false)`
- `const [submitError, setSubmitError] = useState<string | null>(null)`
