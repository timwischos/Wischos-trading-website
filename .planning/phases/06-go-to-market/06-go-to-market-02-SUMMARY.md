## What Was Built

**robots.txt** â€” Added explicit `Allow: /` directives for GPTBot, ClaudeBot, PerplexityBot, anthropic-ai, and ChatGPT-User. These AI crawlers were previously covered by the wildcard `*` allow, but explicit rules prevent ambiguity in newer crawler implementations and signal GEO readiness.

**Blog route JSON-LD** â€” `blog/$slug.tsx` now emits `BlogPosting` structured data including `headline`, `datePublished` (sourced from the new `isoDate` field), `dateModified`, `author`, `publisher`, and `mainEntityOfPage`. This is required for Google to consider blog posts for rich result eligibility and for AI citation systems to attribute the content correctly.

**Gift Sets route JSON-LD** â€” `gift-sets/$setId.tsx` now emits two JSON-LD blocks: `Product` (with SKU, brand, `PriceSpecification` with FOB range, availability, and seller) and `BreadcrumbList` (Home > Gift Sets > [set name]). Mirrors the existing `$productId.tsx` pattern exactly.

**FAQ route** â€” New `/faq` route at `src/routes/{-$locale}/faq.tsx` with 8 company-level buyer Q&As covering MOQ, set contents, logo methods, production timeline, payment terms, packaging, shipping geographies, and QC process. Emits `FAQPage` JSON-LD using the `faqItems` array. FAQ link added to SiteFooter Company column.

**Privacy route** â€” Added `buildOgMeta` + `buildCanonical` to `privacy.tsx` head(), bringing it in line with every other route. Previously emitted only `title` and `description` meta tags with no canonical or OG/Twitter data.

**Organization JSON-LD improvements** â€” Updated `__root.tsx` Organization schema: `areaServed` changed from the string `'Worldwide'` to the array `['AU', 'EU', 'SG', 'KR', 'JP']` matching business priority markets; `foundingDate: '2024'` added; `availableLanguage: 'English'` added inside `contactPoint`.

- **Found during:** Task 3
- **Issue:** `npx tsr generate` returned "No files matched" â€” TanStack Start regenerates the route tree at dev/build time via the Vite plugin, not via a standalone CLI command. - **Files modified:** src/routeTree.gen.ts
